Descripción del taller

En el taller se verán los conceptos fundamentales de cómo aplicar técnicas de Machine Learning para resolver problemas específicos de Ciberseguridad. Estos problemas van desde clasificar malware como lo hace un antivirus, hasta detectar comportamientos sospechosos o intrusiones en una red.

Duración aproximada: 60 minutos.

Número máximo de plazas: 40 plazas.

Requisitos para participar en el taller: Llevar portátil con R y Rstudio instalado.

Este enlace contiene los materiales que se utilizarán en la sesión: https://www.dropbox.com/sh/662uoy0y1qxf6h9/AACVAzcYwIBZN3y8_Ks9TXzpa?dl=0
