Descripción del taller

Durante esta sesión, el asistente obtendrá las nociones básicas sobre cómo un perfil de seguridad debe introducirse dentro de los flujos de DevOps sin penalizar en ningún momento la entrega continua de software que persigue esta filosofía. Además, conocerá como se integran las tecnologías de contenedores como Docker con las propias aplicaciones desarrolladas así como con el resto de herramientas de automatización de construcción y despliegue.

Duración aproximada: 2 horas.

Número máximo de plazas: 50.

Requisitos para participar en el taller: En cuanto a conocimientos, se aceptan alumnos de todos los niveles, aunque podría ser interesante conocer previamente conceptos como contenedores Docker, pipelines de integración/despliegue continuo, etc.

Se aconseja disponer de Linux, Docker (https://docs.docker.com/install) y Docker Compose, aunque también está permitido ir de oyente.
