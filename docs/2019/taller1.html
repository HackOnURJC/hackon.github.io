Descripción del taller

En este taller se codificarán herramientas relacionadas con la seguridad ofensiva (de forma básica) desde cero, con introducción al lenguaje python, para que sea accesible a todos.

Duración aproximada: 2 horas.

Número máximo de plazas: 50.

Requisitos para participar en el taller: Phyton (3.7) y pip de esta versión.

Este enlace contiene los materiales que se utilizarán en la sesión: https://github.com/g4lk/TallerURJC
