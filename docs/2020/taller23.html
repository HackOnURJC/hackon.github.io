REQUISITOS DEL TALLER

Para este taller se necesitan tres máquinas virtuales:

  1. VM con Kali Linux.
  2. VM con Windows 7 y con los FW desactivados (la desactivación se explicará al comienzo del taller). Aquí instalaremos el universal forwarder de Splunk (www.splunk.com/page/previous_releases/universalforwarder#x86windows). Hay que crearse una cuenta gratuita que luego se necesitará para instalar Splunk.
  3. VM, recomendablemente Ubuntu por si sencilla instalación, con una instalación de Splunk (https://www.splunk.com/en_us/download/splunk-enterprise.html). La cuenta gratuita permite hacer queries durante 60 días e indexar hasta 500MB de logs al día, suficientes para nuestros propósitos.

Duración aproximada: 2 horas.

Número máximo de plazas: 25.
