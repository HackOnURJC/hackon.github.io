Introducción a Windows Internals

16/2/2023 17:00 - 19:00 

Nivel intermedio
Resumen de la actividad

En este taller se explicarán conceptos sobre los Internals del sistema operativo Windows, tales como la interacción entre la capa de usuario y el kernel.

• Conceptos básicos de funcionamiento del kernel de Windows.
• Arquitectura del sistema operativo.
• Aplicaciones a la seguridad de estos conocimientos.

Requisitos para la actividad

Equipo con mínimo 8 GB de RAM para poder virtualizar dos sistemas Windows 10. Se compartirá por drive un enlace con las máquinas virtuales a instalar (preferiblemente en VMWare)

Plazas limitadas: 20 plazas
